FeatureConverter

A base class for feature converters.

Preprocesses Object, e.g. sample or linear transform features. Can be applied to each Object in DataSet. But before doing this FeatureConverter should be trained by some FeatureConverterLearner. This can be usefull for better training of Scorer.

Feature converters which were used during training of Scorer will be added in the result Scorer and consequently must be Serializable.

Note:
In most cases you don't need to create FeatureConverter directly, normally it is the result of some FeatureConverterLearner work.

You can convert your data features in this way:

  • FeatureConverter::apply(const Object& input, Object* output)
    Parameters:
    inputObject to be converted
    outputconverted Object
  • FeatureConverter::apply(const DataSet<TElement>& input, DataSet<TElement>* output)
    Parameters:
    inputDataSet to be converted
    outputconverted DataSet

FakeFeatureConverter

Does nothing with input object

FeatureSampler

Samples features with specified indices.

PerFeatureLinearConverter

Perform linear transformation of each feature. 

features[i] -> factors[i] * features[i] + shifts[i]

NanToZeroConverter

Converts NaN features to zero values.

NanToAverageConverter

Converts NaN features to average values of all non NaN features from other Objects. If some feature is NaN in all Objects this feature will be removed.

NominalToBoolConverter

Converts nominal features to boolean, and move they to the end of features list.

Nominal feature with k values will be converted to k boolean features.

RemoveNominalConverter

Remove all nominal features.

FeatureConverterLearner

A base class for feature converter learners.

Learns input dataset and produces a FeatureConverter. Sometimes before training a Scorer it's usefull to make some preprocessing of object features, e.g. normalize, filter out non significant features, etc. In order to do this one need to add FeatureConverterLearner into the BaseLearner.

If you want to make your own FeatureConverterLearner it's more convinient to inherit from BaseFeatureConverterLearner.

You can get your FeatureConverter in this way:

  • FeatureConverterLearner::learn(const DataSet<TElement>& data_set, bool check_parameters = true)
    Parameters:
    data_setdataset to study
    check_parameterswhether perform Parameterized::checkParameters() before launch
  • FeatureConverterLearner::make()

To add your own FeatureConverterLearner you should:

1) Add your feature converter class inherited from FeatureConverter. Your class must have FeatureConverter::convertFeatureInfo() and FeatureConverter::applyImpl() methods;
2) Add your feature converter learner class inherited from FeatureConverterLearner with type of elements and your feature converter class as parameters. Your class must have FeatureConverterLearner::learnImpl() private method;

See NanToAverageConverterLearner as example.

FeatureNormalizerLearner

Independently normalize each feature component to the specified range.

Example: learn converter and convert features. 
#include <iostream>

#include "ltr/data/object.h"
#include "ltr/data/data_set.h"
#include "ltr/feature_converters/feature_converter.h"
#include "ltr/feature_converters/feature_normalizer_learner.h"
#include "ltr/utility/indices.h"

using std::cout;
using std::endl;

using ltr::Object;
using ltr::DataSet;
using ltr::FeatureConverter;
using ltr::FeatureNormalizerLearner;

using ltr::utility::Indices;
using ltr::utility::IndicesPtr;

int main() {
  Object object1, object2, object3;

  object1 << 5.234 << 1.533 << -5.378;
  object2 << 2.213 << -1.234 << 0.323;
  object3 << -2.123 << 0.207 << -6.085;

  DataSet<Object> data, converted_data;
  data.add(object1);
  data.add(object2);
  data.add(object3);

  FeatureNormalizerLearner<Object> feature_normalizer_learner;
  feature_normalizer_learner.set_min(-1.0);
  feature_normalizer_learner.set_max(1.0);
  feature_normalizer_learner.learn(data);

  FeatureConverter::Ptr converter = feature_normalizer_learner.make();
  converter->apply(data, &converted_data);

  cout << "Before:" << endl << data << endl;
  cout << "After:" << endl << converted_data << endl;

  return 0;
}
Output: 
Before:
[5.23, 1.53, -5.38]{0.00, 0.00}
[2.21, -1.23, 0.32]{0.00, 0.00}
[-2.12, 0.21, -6.08]{0.00, 0.00}

After:
[1.00, 1.00, -0.78]{0.00, 0.00}
[0.18, -1.00, 1.00]{0.00, 0.00}
[-1.00, 0.04, -1.00]{0.00, 0.00}

FeatureSamplerLearner

Produces FeatureSampler with specified indices.

Example: learn converter and convert features. 
#include <iostream>

#include "ltr/data/object.h"
#include "ltr/data/data_set.h"
#include "ltr/feature_converters/feature_converter.h"
#include "ltr/feature_converters/feature_sampler_learner.h"
#include "ltr/utility/indices.h"

using std::cout;
using std::endl;

using ltr::Object;
using ltr::DataSet;
using ltr::FeatureConverter;
using ltr::FeatureSamplerLearner;

using ltr::utility::Indices;
using ltr::utility::IndicesPtr;

int main() {
  Object object1, object2, object3;

  object1 << 5.234 << 1.533 << -5.378;
  object2 << 2.213 << -1.234 << 0.323;
  object3 << -2.123 << 0.207 << -6.085;

  IndicesPtr indices(new Indices());
  indices->push_back(0);
  indices->push_back(2);

  DataSet<Object> data, converted_data;
  data.add(object1);
  data.add(object2);
  data.add(object3);

  FeatureSamplerLearner<Object> feature_sampler_learner;
  feature_sampler_learner.set_indices(indices);
  feature_sampler_learner.learn(data);
  FeatureConverter::Ptr converter = feature_sampler_learner.make();

  converter->apply(data, &converted_data);

  cout << "Before:" << endl << data << endl;
  cout << "After:" << endl << converted_data << endl;

  return 0;
}
Output: 
Before:
[5.23, 1.53, -5.38]{0.00, 0.00}
[2.21, -1.23, 0.32]{0.00, 0.00}
[-2.12, 0.21, -6.08]{0.00, 0.00}

After:
[5.23, -5.38]{0.00, 0.00}
[2.21, 0.32]{0.00, 0.00}
[-2.12, -6.08]{0.00, 0.00}
