Detailed Description

template<typename T>
class ltr::Vectorized< T >

Vector for inheritance in classes. It provides methods of std::vector for custom classes. Usage: inherit your class from Vectorized<your_type> and add Vectorized<your_type>(your_vector_name) call in all your constructors to associate your_vector_name vector with vector methods.
