Detailed Description

Class for Rosenbrock function in \(\mathbb{R}^n\).

\(f(x) = \sum_{i = 1}^{n - 1} (100 * (x_i^2 - x_{i + 1}) + (x_i - 1)^2)\)
Minimum at \( x = (1, \dots, 1), \quad f(x) = 0 \)
