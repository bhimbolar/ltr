Detailed Description

A composition scorer - contains other scorers with their weights. Scores as weighted order statistic of outputs of its weak scorers. Has a "RATIO" parameter, if ratio = 0 then it scores the minimum of weak scores, if ratio = 1 - the maximum, if ratio = 0.5 - the weighted median. By default ratio = 0.5
