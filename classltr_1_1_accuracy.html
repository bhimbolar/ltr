Detailed Description

template<class TElement>
class ltr::Accuracy< TElement >

Accuracy measure. Can be poinwise and pairwise, but not listwise. Returns 1 if actual score is equal (almost) to predicted one (or for pairs if actual and predicted pair orders are equal) and 0 otherwise
