How to create new learner and scorer

Table of Contents

Creating new scorer

It is necessary to create a scorer before creating a learner.
First of all you should create a scorer class inherited from Scorer abstract class. Let's call this class "NewScorer"
It's recommended to create a typedef for the shared pointer: 

typedef ltr::utility::shared_ptr<NewScorer> Ptr;

Then you should decide which field your scorer will content and create them.
After this create constructors.
Now you should create method scoreImpl, that has signature, written in the code below: 

virtual double scoreImpl(const Object& object) const

scoreImpl is the most important method of the class. It implements scoring of the recieved object.

Also NewScorer should contain method generateCppCodeImpl with such signature: 

virtual string generateCppCodeImpl(const string& object_name) const;

This function will serialize object of NewClass into valid C++ code.

Creating new learner

Now if you have created a NewScorer class, you can start to create an own learner. Let's call this class "NewLearner"
This class must be inharited from BaseLearner. First of all you should create a typedef for the shared pointer: 

typedef ltr::utility::shared_ptr<NewLearner> Ptr;

Then create fields that are necessary for the NewLearner. You should create at least two constructors.
First container will recieve ParametersContainer and the second will recieve explicit parameters that will initialize field of the NewClass.

After creating all this stuff, you can start to create the most important method of the learner - learnImpl: 

void learnImpl(const DataSet<TElement>& data, NewScorer* scorer)

As you see, this function recieves learning data set and pointer to NewScorer.
After the learning process this method must generate a NewScorer.
Then you can use generated scorer to predict the labels of objects.
