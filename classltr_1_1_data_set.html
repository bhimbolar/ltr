Detailed Description

template<typename TElement>
class ltr::DataSet< TElement >

Template class that implements DataSet.

Template Parameters:
Typeof elements, which would be stored in the DataSet. The elements should implement Object container interface. Those could be Object, ObjectPair or ObjectList.
