Detailed Description

template<typename ObjectType, typename ScorerType>
class ltr::cv::CrossValidator< ObjectType, ScorerType >

Performs crossvalidation and forms a printable set of tables called multitable.

Info about cross-validation can be found here: http://en.wikipedia.org/wiki/Cross-validation_(statistics)

Multitable in CrossValidator contains 4 dimentions: -learners -splitters -measures -datasets

To perform crossvalidation add to the crossvalidator datasets, measures, splitters, learners. Every entity must be contained in crossvalidtor at least once. After the initialization is done, perform "launch()" command.

Crossvalidator is printable
