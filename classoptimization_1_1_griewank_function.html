Detailed Description

Class for Griewank function in \(\mathbb{R}^n\).

\(f(x) = \sum_{i = 1}^n x_i^2/4000 - \prod_{i = 1} ^ n \cos(x_i / \sqrt(i))\), minimum at \( x = (0, \dots, 0), \quad f(x) = 0 \).

Get more info on http://www-optima.amp.i.kyoto-u.ac.jp/member/student/hedar/Hedar_files/TestGO_files/Page1905.htm
