Detailed Description

template<class TElement>
class ltr::DataPreprocessor< TElement >

A base class for data preprocessors.

Preprocesses DataSet, e.g. delete or replicate elements in DataSet. This can be usefull for better learning (see Learner<TElement>::addDataPreprocessor ). Should not change Object features and FeatureInfo as it won't be saved in Scorer. Use FeatureConverter for feature convertions.

See also:
FeatureConverter, BaseLearner
