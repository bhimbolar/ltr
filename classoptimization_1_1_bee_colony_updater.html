Detailed Description

template<typename TFunction>
class optimization::BeeColonyUpdater< TFunction >

Class for update points from population in the artificial bee colony algorithm.

Get more information about the updater on http://sci2s.ugr.es/EAMHCO/pdfs/ABC-algorithm-numerical-function-2007.pdf
