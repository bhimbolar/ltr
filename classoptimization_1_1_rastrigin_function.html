Detailed Description

Class for Rastrigin function in \(\mathbb{R}^n\).

\(f(x) = 10 n + \sum_{i=1}^n \left(x_i^2 - 10\cos(2 \pi x_i)\right)\)
Minimum at \( x = (0, \dots, 0), \quad f(x) = 0 \).
Get more info on http://www-optima.amp.i.kyoto-u.ac.jp/member/student/hedar/Hedar_files/TestGO_files/Page2607.htm
