Detailed Description

Class for Easom function in \(\mathbb{R}^2\).

\(f(x, y) = - e^{-(x - \pi)^2} * e^{-(y - \pi)^2} * \cos(x) * \cos(y)\) \(f(x, y) = - g(x) * g(y), \text{where } g(x) = e^{-(x - \pi)^2} * cos(x)\), minimum at \( (x, y) = (\pi, \pi), \quad f(x, y) = -1\).

Get more info on http://www-optima.amp.i.kyoto-u.ac.jp/member/student/hedar/Hedar_files/TestGO_files/Page1361.htm.
