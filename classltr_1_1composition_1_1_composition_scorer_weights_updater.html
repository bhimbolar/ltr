Detailed Description

template<class TElement>
class ltr::composition::CompositionScorerWeightsUpdater< TElement >

Is a parameter for composition learner. Implements a strategy of updating composition's weights while adding a new scorer to it. E. g. simple boosting always updates only the weight of the last scorer, whereas gradient boosting updates all weights in composition
