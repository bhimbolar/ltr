Bayesian classifiers are simple classification algorithms, that use the idea of probability density.

Such algorithms try to find a probability of every class using training dataset. And then choose the class with the highest probability for classifiable object.

To learn more visit http://en.wikipedia.org/wiki/Bayesian_inference

Density Estimation

You can estimate probability density of some particular DataSet with LTR.

First of all choose the estimation algorithm (Naive Bayes, QDA or Fisher LDA).

After this, declare required probability density learner (NormalNaiveBayesDensityLearner, QuadraticDiscriminantDensityLearner respectively or FisherDiscriminantDensityLearner).

Then learn declared estimation learner on some dataset using ::learn() method and create a BaseProbabilityDensityEstimator object using ::make() method.

Now you can use generated BaseProbabilityDensityEstimator for probability density estimation

List of all density estimation learners

Density estimation learner Description
NormalNaiveBayesDensityLearner Assumes the independence of features
QuadraticDiscriminantDensityLearner Assumes that the measurements from each class are normally distributed
FisherDiscriminantDensityLearner Assumes that the resulting combination may be used as a linear classifier

Bayesian Classification

LTR allows to use different algorithms of unparameterized Bayesian classification such as Naive Bayes, Quadratic Discriminant and Fisher Discriminant.

To implement Bayesian Classification was used common approach: there is a class BayesianLearner that recieves DataSet and generates BayesianScorer. BayesianScorer can be used to score new objects.

Known, that the main idea of Bayesian Classification is the estimation of the probability density and after this choosing the class depending on probabilities for every classifiable object.

The main field of BayesianLearner is estimator_. It keeps the pointer to probability density estimator and use it to classify objects.

To use Bayesian Learner you need to:

  • choose estimation algorithm
  • according to chosen algorothm, declare either FisherDiscriminantLearner or NormalNaiveBayesLearner or QuadraticDiscriminantLearner
  • learn the declared classifier on some DataSet with ::learn() method
  • generate classifier with ::make() method
Example: usage of LTR Bayesian classification Output:
#include <iostream>

#include "ltr/data/object.h"
#include "ltr/data/object_pair.h"
#include "ltr/data/object_list.h"
#include "ltr/data/data_set.h"
#include "ltr/density_estimators/base_probability_density_learner.h"
#include "ltr/density_estimators/fisher_discriminant_density_learner.h"
#include "ltr/density_estimators/fisher_discriminant.h"
#include "ltr/learners/bayesian_learner/bayesian_learner.h"
#include "ltr/scorers/bayesian_scorer.h"

using std::cout;
using std::endl;

using ltr::Object;
using ltr::DataSet;
using ltr::BayesianScorer;
using ltr::BayesianLearner;
using ltr::FisherDiscriminant;
using ltr::FisherDiscriminantLearner;
using ltr::FisherDiscriminantDensityLearner;
using ltr::BaseProbabilityDensityLearner;

int main() {
  Object object1;
  object1 << 0 << 0;
  object1.set_actual_label(1.0);

  Object object2;
  object2 << 10 << 0;
  object2.set_actual_label(0.0);

  Object object3;
  object3 << 12 << 0;
  object3.set_actual_label(0.0);

  DataSet<Object> data;
  data.add(object1);
  data.add(object2);
  data.add(object3);
  FisherDiscriminantLearner<Object> learner;
  learner.learn(data);
  BayesianScorer::Ptr scorer = learner.makeSpecific();

  Object object4;
  object4 << 1 << 0;
  cout << scorer->score(object4) << endl;
}
